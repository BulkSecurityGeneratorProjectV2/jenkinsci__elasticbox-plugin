A version of the selected box to launch slave. The box version or one of its child boxes must have text variables JENKINS_URL and JNLP_SLAVE_OPTIONS. During the launch, ElasticBox Jenkins plugin will set the value of JENKINS_URL variable to the Jenkins URL configured in the global system configuration and the value of JNLP_SLAVE_OPTIONS to the commend line options that the slave box can use to start the JNLP slave agent in the slave.

If Latest is selected then the version will be the box itself or the last version depending on how the box is shared in the selected workspace.
